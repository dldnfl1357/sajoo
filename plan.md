# 🔮 사주포춘 - AI 사주 분석 서비스 기획서

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 서비스명 | 사주포춘 (SajuFortune) |
| 컨셉 | 생년월일시 기반 사주팔자 분석 웹 서비스 |
| 목표 | 2시간 내 MVP 완성, 즉시 공유 가능한 형태 |
| 핵심 가치 | 재미 + 공유성 + 접근성 |

---

## 2. 타겟 사용자

- **1차 타겟**: 20-40대 운세/사주에 관심 있는 일반인
- **사용 시나리오**:
  - 새해/생일에 한 해 운세 확인
  - 친구들과 결과 공유하며 재미 요소로 활용
  - 중요한 결정 전 참고용으로 확인

---

## 3. 핵심 기능 (MVP)

### 3.1 입력 정보
```
✅ 이름 (선택)
✅ 생년월일 (필수) - 양력/음력 선택
✅ 태어난 시간 (선택) - 모르면 "모름" 선택 가능
✅ 성별 (선택)
```

### 3.2 분석 결과 제공
```
📌 사주팔자 (四柱八字)
   - 년주, 월주, 일주, 시주 표시
   - 천간/지지 한자 + 한글 병기

📌 오행 분석
   - 목/화/토/금/수 비율 시각화
   - 강한 기운 / 부족한 기운 안내

📌 타고난 성격
   - 일간(日干) 기반 성격 분석
   - 장점 / 단점 / 적성

📌 2025년 운세
   - 올해의 전체 운세 요약
   - 재물운 / 애정운 / 건강운 / 직장운

📌 조언 한마디
   - AI가 생성하는 맞춤형 조언
```

### 3.3 부가 기능
```
🔗 결과 공유하기 (링크 복사 / 카카오톡)
📸 결과 이미지 저장 (캡처용 카드 UI)
🔄 다시 보기
```

## 4. 사용자 플로우

```
[시작화면]
    │
    ▼
[정보 입력] ─── 이름, 생년월일, 시간, 성별
    │
    ▼
[로딩 애니메이션] ─── "운명을 분석하고 있습니다..." (2-3초)
    │
    ▼
[결과 화면]
    ├── 사주팔자 표시
    ├── 오행 차트
    ├── 성격 분석
    ├── 2025 운세
    └── 공유 버튼
    │
    ▼
[공유/재시작]
```

## 6. 기술 스택

| 구분 | 기술 | 선택 이유 |
|------|------|----------|
| 프론트엔드 | React (단일 파일) | 빠른 개발, 인터랙티브 UI |
| 스타일링 | Tailwind CSS | 빠른 스타일링 |
| 사주 계산 | JavaScript 로직 | 만세력 알고리즘 구현 |
| AI 해석 | Claude API | 자연스러운 운세 해석 |
| 배포 | Artifact | 즉시 확인 가능 |

## 8. 비즈니스 확장 가능성

### 단기 (출시 후)
- 카카오톡/인스타그램 공유 기능
- 결과 이미지 다운로드

### 중기 (1-3개월)
- 궁합 보기 (두 사람 비교)
- 월별/주별 상세 운세
- 유료 프리미엄 상세 분석

### 장기 (3-6개월)
- 회원가입 + 운세 알림
- 타로/별자리 등 콘텐츠 확장
- 광고 수익 모델
